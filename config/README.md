Die Termine können folgende Angaben enthalten.

* Datum im Format YYYY-MM-DD
* Zeit im Format HH:MM
* Titel
* URL für Titel (optional)
* Ort
* URL für Ort (optional)
* Beschreibung (optional)

Beispiel:

```
date=2015-09-19
time=20:00
title=Was Ihr Wollt!
title.url=
location=Varia Vineta
location.url=
description=
```
